WLWCCS Hosting

WLWCCS Hosting is a free Minecraft server hosting service that provides free 24/7 server runtime.

WLWCCS Hosting is a relatively simple service - but when needed, here is a small guide to get started! Create your server by signing up - everything from there should be pretty straightforward. Questions about server limitations and such are addressed in the FAQ. If you wish to contribute, there's a link to the GitHub below. A proper guide is shown in the documentation on the codebase.
