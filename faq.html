FAQ

Q: What are the server limitations?

All servers can only hold a total of 10 players at once. There isn't nessecarily a limit to how much RAM or CPU percentage is dedicated to a server, but the amount of each spec a server receives is readjusted over time based on the overall status regarding the other servers currently running.

Q: What other limitations are there outside of the servers?

You can only have one server per account. And before you ask - no, it is not permitted to create alternative accounts to run multiple servers at once.

Q: If WLWCCS is free - how do we make revenue?

WLWCCS Hosting will utilise ad revenue to make up for the free server hosting we provide.

Q: How do I actually access servers in Minecraft?

Bedrock Edition

When you go into the servers section of Minecraft, scroll down and you should find a button that says "Add Server". Click on it, and you'll be prompted for the nickname you'll give the server and the server credentials. The port number is given on the server panel. The IP address is wlwccs.duckdns.org. Then you should be able to add the server and play on it.

Java Edition

Enter the Multiplayer section on the main menu and you'll find the "Add Server" option - similar to Bedrock. You will then be prompted with a server address field and a nickname field. The nickname field is up to you - this is completely cosmetic. This applies to Bedrock as well. The server address field requires you to write the IP address and port in the same field - split by a colon. This should be structured as "wlwccs.duckdns.org:". Obviously replace with the port on your panel. Press done and you should be able to access the server from there.
